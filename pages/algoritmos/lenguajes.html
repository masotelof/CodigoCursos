Introducción a los Lenguajes de Programación

Introducción

Los Programas de Computadora son una secuencia de instrucciones que permiten a las computadoras procesar datos. Dichos programas le especifican a la computadora las acciones que debe de realizar; estos son realizados por los Programadores en un determinado Lenguaje de Programación y se le conoce como el Software de la computadora.

La velocidad con la que se ejecuta un Programa depende de el número de operaciones de punto flotante que puede ejecutar el microprocesador de una computadora (llamados también FLOPS por sus siglas en ingles de Floating Point Operations Per Seconds). La velocidad de ejecución de un programa sigue la Ley de Moore, la cual menciona que cada uno o dos años la velocidad y memoria de un computadora se duplica.

Tipos de Lenguajes de Programación

Los Lenguajes de Programación se clasifican en tres tipos, esto dependiendo de los pasos necesarios para que puedan ser interpretados por la Computadora.

Lenguaje Máquina

Es aquel que puede ser interpretado por el microprocesador de una computadora, este esta definido por la arquitectura del microprocesador y el conjunto de instrucciones soportadas por el mismo. Este lenguaje esta conformado por una serie de 0 y 1.

Lenguaje Ensamblador

Es un conjunto de abreviaturas en ingles que sirven para representar operaciones básicas. Este lenguaje surge de la complejidad de escribir programas en Lenguaje Maquina. Junto con el Lenguaje Ensamblador fue necesario desarrollar un programa Ensamblador el cual es el encargado de convertir el código escrito en Lenguaje Ensamblador a Lenguaje Máquina.


section     .text
global      _start                  ;must be declared for linker (ld)

_start:                             ;tell linker entry point

mov     edx,len                   ;message length
mov     ecx,msg                   ;message to write
mov     ebx,1                     ;file descriptor (stdout)
mov     eax,4                     ;system call number (sys_write)
int     0x80                      ;call kernel

mov     eax,1                     ;system call number (sys_exit)
int     0x80                      ;call kernel

section     .data

msg     db  'Hello, world!',0xa     ;our dear string
len     equ $ - msg                 ;length of our dear string

Lenguaje de Alto Nivel

Es el que permite escribir instrucciones que parecen oraciones en Idioma Ingles, también permite introducir una serie de expresiones matemáticas. Para convertir del Lenguaje de Alto Nivel a Lenguaje Maquina es necesario el uso de un programa llamado Compilador; esto puede llevar un tiempo considerable. Además existen programas denominados Interpretes los cuales permiten ejecutar un programa escrito en Lenguaje Máquina de manera directa sin necesidad de compilarlo; a este tipo de lenguajes se les llama Lenguaje de Script y por lo mismo son más lentos que los compilados, algunos de estos lenguajes son: Javascript, PHP, Python, Perl, etc.
